Full Stack Web Development

Our full stack web development program will help you to build modern fully-functional web applications, from the visual aspects rendered in the browser all the way through the application logic on the server, using best practises of software and web development along the way. Program Details

  • Self-paced, online learning
  • Intermediate skill level
  • 450 total hours
  • 250 CHF per month
  • Personalized support
  • Up to 15 ECTS credits
